Pytorch-Lightning Implementation of RandLA-Net && SQN
Nana Wang
2023/10
code1 / code2

RandLA-Net(CVPR2020) is a light net which can process efficient semantic segmentation for 3D point clouds, and SQN(ECCV2022) is based on RandLA-Net and achieves promising performance on seven large-scale open datasets under weak supervision schemes. The implementation of RandLA-Net(CVPR2020) is in code0 (for large-scale semantic segmentation). and SQN(ECCV2022) is in code1 (for weak supervision schemes).
